## 프로젝트 개요
학생들은 여러 과목을 공부해야 하지만 **무엇을 먼저, 얼마나 공부해야 할지** 계획 세우는 것 자체가 부담입니다.

AutoPlanner는 사용자의 공부 성향과 과목/챕터 정보를 입력하면 LLM이 최적화된 학습 계획표를 생성해주고, Notion Calendar에 바로 연동할 수 있는 서비스입니다.
